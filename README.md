## 주제 Ezen_newrent
> 렌트카 예약 관리 시스템 

## 프로젝트 소개
사용자와 관리자의 서비스를 기반으로 한 렌트카 예약 관리 시스템을 구축할 것 입니다. 
관리자와 사용자간의 편리한 서비스를 구축하기위해 차량관리, 예약관리, 사용자와 관리자간의 소통시스템을 위주로 설계하였습니다.
